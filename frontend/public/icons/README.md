# Ícones do PWA

Este diretório deve conter os seguintes ícones para o PWA:

## Ícones Necessários

1. **icon-192x192.png** (192x192 pixels)
   - Ícone padrão para Android e PWA
   
2. **icon-512x512.png** (512x512 pixels)
   - Ícone de alta resolução para Android e PWA

3. **icon-maskable-192x192.png** (192x192 pixels)
   - Ícone maskable para Android (deve ter padding seguro de 10% em todos os lados)
   - O conteúdo importante deve estar dentro do círculo central de 80% do tamanho

4. **icon-maskable-512x512.png** (512x512 pixels)
   - Ícone maskable de alta resolução para Android

## Como Criar os Ícones

Você pode usar ferramentas como:
- [PWA Asset Generator](https://github.com/elegantapp/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Design tools como Figma, Photoshop, etc.

## Formato

- Formato: PNG com fundo transparente ou sólido
- Recomendado: Fundo sólido com cores do tema (#4f46e5 ou #0f1b2e)
- Para maskable icons: Certifique-se de que o conteúdo importante está dentro de um círculo central de 80% do tamanho total

## Placeholder

Por enquanto, você pode usar qualquer ícone 192x192 e 512x512 como placeholder até criar os ícones oficiais.
